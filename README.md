# Sistema de Locadora de Veículos

## Sobre o projeto

Este projeto foi desenvolvido em Java com o objetivo de consolidar os conhecimentos de Programação Orientada a Objetos (POO), aplicando conceitos como herança, abstração, encapsulamento, associação, tratamento de exceções e manipulação de coleções.

O sistema simula o gerenciamento de uma locadora de veículos através de um menu interativo no console.

---

## Funcionalidades

### Clientes
- Cadastro de clientes
- Listagem de clientes
- Atualização de dados
- Remoção de clientes

### Veículos
- Cadastro de veículos
- Listagem de veículos
- Atualização de dados
- Remoção de veículos
- Controle de disponibilidade

### Vendedores
- Cadastro de vendedores
- Listagem de vendedores
- Atualização de dados
- Remoção de vendedores
- Registro de locações realizadas

### Locações
- Realizar locação
- Finalizar locação
- Listar locações
- Controle de locações ativas

---

## Regras de negócio

O sistema possui diversas validações para garantir a integridade dos dados, como:

- Não permite cadastrar clientes com CPF duplicado.
- Não permite cadastrar veículos com placa duplicada.
- Não permite cadastrar saldo negativo.
- Não permite cadastrar veículos com preço inválido.
- Não permite alugar veículos indisponíveis.
- Não permite finalizar uma locação inexistente.
- Não permite remover clientes com locações ativas.
- Não permite remover vendedores responsáveis por locações ativas.
- Não permite remover veículos que estejam alugados.
